# Foot & Ankle Page v2 Upload Instructions

This v2 package only changes the Foot & Ankle page and adds the required Foot & Ankle images plus Convergence Health logo assets.

## Upload / replace these files

1. Replace the root-level `foot-ankle.html` with the file in this zip.
2. Upload the full folder: `assets/photos/foot-ankle/`
3. Upload the logo files in: `assets/logos/`

## Do not replace

This package does not include or require replacing `assets/css/styles.css`. The page uses the existing website stylesheet and includes its own page-specific inline CSS so the rest of the website remains untouched.

## What changed in v2

- Replaced the placeholder Convergence logo with the supplied official logo assets.
- Added the Lake Havasu return-to-activity image.
- Added the crutches / fracture recovery image.
- Strengthened the scope clarification that Dr. Barnes does not provide routine podiatry care: toenails, calluses, diabetic foot care, bunions or hammertoes.
- Kept photo use unique: no patient photos are intentionally duplicated on the page.
- Added stronger condition cards, treatment carousel, scope section and FAQ content.
