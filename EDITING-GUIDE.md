# James Barnes, MD Website Editing Guide — v23 Organized Build

This build reorganizes the site so future edits can be made without replacing everything.

## Main files
- `index.html` — homepage
- `shoulder.html`, `hip.html`, `knee.html`, `hand.html`, `sports-medicine.html`, `orthobiologics.html` — condition/service pages
- `assets/css/styles.css` — all site styling
- `assets/js/` — reserved for future JavaScript modules

## Upload workflow
For the v23 reorganization, delete old website files/folders except your repository README, then upload the contents of this ZIP.

After v23, future updates can usually be partial uploads:
- Change colors/layout: upload only `assets/css/styles.css`
- Change homepage: upload only `index.html`
- Change photos: upload only the relevant file in `assets/photos/...`
- Change Orthobiologics: upload only `orthobiologics.html` and any changed graphics
